# PriorityBoard

A beautiful, dark-themed priority management board to organize your work priorities by category with drag & drop ranking.

![PriorityBoard](https://img.shields.io/badge/Status-Live-brightgreen)

## Features

- **4 Preset Categories** — Work, Personal, Health, Learning
- **Drag & Drop Ranking** — Reorder priorities within or across columns
- **Due Dates** — With overdue indicators
- **Notes** — Expandable notes per card
- **Completion Tracking** — Check off items with satisfying animations
- **Progress Dashboard** — Stats, category progress bars, overdue list, and streak counter
- **Persistent** — All data saved in localStorage, nothing leaves your browser
- **Zero Dependencies** — Single HTML file, no build step needed

## Tech Stack

- Vanilla HTML/CSS/JS
- [SortableJS](https://sortablejs.github.io/Sortable/) (CDN) for drag & drop
- localStorage for persistence
- GitHub Actions for CI/CD deployment to GitHub Pages

## Getting Started

Open `index.html` in any browser, or visit the live deployment on GitHub Pages.

## Deployment

Automatically deployed via GitHub Actions on every push to `main`.
